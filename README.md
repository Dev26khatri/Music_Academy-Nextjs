# music_academy-nextjs

This is a **static frontend project** built using **Next.js** and the **Aceternity UI library**.  
The project is focused on creating a modern and smooth user interface for a music academy.

## Features

- Static frontend project
- Modern UI using Aceternity components
- Clean and simple design
- Smooth animations

## Tech Stack

- Next.js
- React
- Aceternity UI
- Tailwind CSS

## Purpose

This project is created for learning and practicing frontend development.  
There is no backend or database used in this project.

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```
2. Run the project

   ```bash
   npm run dev
   ```

3. Open your browser
   ```bash
   http://localhost:3000
   ```
